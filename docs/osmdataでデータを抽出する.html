5 osmdataでデータを抽出する

5.1 はじめに

osmdataは、open street mapのデータを抽出できるPackageです。 簡易に分析するときなど、osmdataを使うのもありかなと…。

5.2 Package

osmdataです。GIS系のsfも使います。leafletはアウトプット用。

library(leaflet)
library(osmdata)
library(sf)

5.3 データを取得する範囲を決める

まずは範囲を決めます。今回は、仙台市にします。 これは、仙台市がすっぽり入るBox範囲内を対象にしています。

read_bb = getbb('sendai')
print(read_bb)
##         min      max
## x 140.46947 141.2603
## y  38.05085  38.4547

5.4 レジャーをゲット

範囲内にあるレジャーなところを取得します。 PC環境によりますが、データ取得には数分かかります。

x = opq(bbox = read_bb) %>% 
  add_osm_feature(key = 'leisure') %>% 
  osmdata_sf()

#polygonだけにする。
x_sf = x$osm_polygon
x_sf = subset(x_sf,!is.na(x_sf$name))

#余計な列を除外。
x_sf_disp = x_sf[,c("name","geometry")]
print(x_sf_disp)

5.5 地図にプロット

レジャーなポリゴンを表示します。どこに行こうかな。

#mapに表示する
leaflet() %>% 
  addTiles() %>%
  addPolygons(data= x_sf_disp, color = "blue", popup=~name)
