14 Rasterの基本1

14.1 Package

今現在は、「terra」です。

library(sf)
library(terra)
library(ggplot2)

14.2 ラスターの作成

ラスターは、以下のように作成することができます。

#Create 5x5 raster
raster_1 = rast(
  ncol=5,nrow=5,
  xmin=0, xmax=5, ymin=0, ymax=5
)
# raster_1

values(raster_1) = c(1:ncell(raster_1))
raster_1
## class       : SpatRaster 
## dimensions  : 5, 5, 1  (nrow, ncol, nlyr)
## resolution  : 1, 1  (x, y)
## extent      : 0, 5, 0, 5  (xmin, xmax, ymin, ymax)
## coord. ref. : lon/lat WGS 84 (CRS84) (OGC:CRS84) 
## source(s)   : memory
## name        : lyr.1 
## min value   :     1 
## max value   :    25
#output
plot(raster_1,main="raster_1")
text(raster_1,col="white")

14.3 ラスターの作成2

続いて、もう一つのラスターを作成します。値はランダムに。

#Create 5x5 raster
raster_2 = rast(
  ncol=5,nrow=5,
  xmin=0, xmax=5, ymin=0, ymax=5
)
# raster_1

values(raster_2) = sample(ncell(raster_2))
raster_2
## class       : SpatRaster 
## dimensions  : 5, 5, 1  (nrow, ncol, nlyr)
## resolution  : 1, 1  (x, y)
## extent      : 0, 5, 0, 5  (xmin, xmax, ymin, ymax)
## coord. ref. : lon/lat WGS 84 (CRS84) (OGC:CRS84) 
## source(s)   : memory
## name        : lyr.1 
## min value   :     1 
## max value   :    25
#output
plot(raster_2,main="raster_2")
text(raster_2,col="white")

14.4 ラスターの合計

ラスター同士は、合計だってできます。

r_sum = raster_1 + raster_2

#output
plot(r_sum,main="r_sum")
text(r_sum,col="white")
