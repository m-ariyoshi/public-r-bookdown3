7 シーケンス図を描く

7.1 はじめに

20年位前、こういう図はEXCELで描いていました。 まぁ、いまでも時々使いますが、こういうのも再現可能に作成できるのは本当にうれしいです。

7.2 Package

使用するのは、「DiagrammeR」パッケージです。

library(DiagrammeR)

7.3 シーケンス図を書く

こんな感じに、シーケンス図が出来上がります。ALTにも対応している。

# Create a sequence diagram
DiagrammeR("
sequenceDiagram;
   customer->>ticket seller: ask for a ticket;
   ticket seller->>database: seats;
   alt tickets available
     database->>ticket seller: ok;
     ticket seller->>customer: confirm;
     customer->>ticket seller: ok;
     ticket seller->>database: book a seat;
     ticket seller->>printer: print a ticket;
   else sold out
     database->>ticket seller: none left;
     ticket seller->>customer: sorry;
   end
")

7.4 シーケンス図を書く2

こんなのもできる。

DiagrammeR("
sequenceDiagram;
    participant Alice
    participant Bob
    Alice->>John: Hello John, how are you?
    loop HealthCheck
        John->>John: Fight against hypochondria
    end
    Note right of John: Rational thoughts <br/>prevail!
    John-->>Alice: Great!
    John->>Bob: How about you?
    Bob-->>John: Jolly good!
")

7.5 よく書くやつ

私の世界では、こんな図をよく書いていた。

DiagrammeR("
sequenceDiagram;
    participant UE
    participant eNB
    UE->>eNB: rrcConnectionRequest
    Note right of eNB: Admission Check
    alt Admission OK
     eNB->>UE: rrcConnectionSetup
     UE->>eNB: rrcConnectionSetupComplete
    else Admission NG
     eNB->>UE: rrcConnectionReject
   end

")
