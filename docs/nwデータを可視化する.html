9 NWデータを可視化する

9.1 はじめに

FromとToがあるデータ（AからBへValueを渡す。みたいなデータ）がある場合の可視化の例です。

9.2 使用するデータ

今回はデータを作成します。

a_group = c("A","B","C","D","E")

nw_data = data.frame(
  # id = seq(1:5),
  "From_ID" = sample(a_group,5),
  "To_ID"   = sample(a_group,5),
  "Value"   = sample(1:10,5, replace=F)
)
head(nw_data)
##   From_ID To_ID Value
## 1       B     D    10
## 2       D     B     8
## 3       A     A     2
## 4       E     E     5
## 5       C     C     9

9.3 Package

おすすめは、「visNetwork」です。

library(visNetwork)

9.4 Nodeの作成

Nodeを作成します。

#nodeデータの作成
nodes = data.frame(
  id = a_group,
  label = a_group
)
head(nodes)
##   id label
## 1  A     A
## 2  B     B
## 3  C     C
## 4  D     D
## 5  E     E

9.5 edgesの作成

edgesを作成します。

edges = data.frame(
  from = nw_data$From_ID,
  to   = nw_data$To_ID,
  label = as.character(nw_data$Value),
  width = nw_data$Value/2,
  color = "darkgray"
)
head(edges)
##   from to label width    color
## 1    B  D    10   5.0 darkgray
## 2    D  B     8   4.0 darkgray
## 3    A  A     2   1.0 darkgray
## 4    E  E     5   2.5 darkgray
## 5    C  C     9   4.5 darkgray

9.6 可視化する

nodeとedgeが出来上がれば、可視化できます。グリグリ動きます。

visNetwork(nodes, edges) %>%
  visEdges(arrows = 'to', length = 200)
