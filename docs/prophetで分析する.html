11.3 Package

おすすめは、「Prophet」です。 ARIMAとかSARIMAとかあるようですが、難しそうで…。

library(tidyverse)
library(prophet)

11.4 入力データの準備

prophetで分析するために、列名を整えます。

#change
read_df$Date = as.Date(read_df$Date)
colnames(read_df) = c("ds","y")

11.5 prophet分析:モデルの作成

prophetでは、まずモデルを作成します。 パラメータは、適当に。

p_model = read_df %>% prophet(
  growth = "linear",
  # changepoints = NULL,
  # n.changepoints = 25,
  # changepoint.range = 0.8,
  changepoint.prior.scale = 0.05,

  seasonality.mode = "multiplicative",
  seasonality.prior.scale = 10,
  
  yearly.seasonality = TRUE,
  weekly.seasonality = TRUE,
  daily.seasonality  = FALSE,
  
  holidays = NULL,
  holidays.prior.scale = 10
)

11.6 prophet分析:将来の定義

将来を定義します。ここでは７日間にしました。

future <- make_future_dataframe(p_model, periods = 7)

11.7 prophet分析:モデルから将来を予測

forecast <- predict(p_model, future)

11.8 prophet分析:結果を出力１

dygraphで簡単に出力までできます。

dyplot.prophet(p_model, forecast)

11.9 prophet分析:結果を出力２

季節性とか、これで確認もできます。 ８月と１月が増えるトレンドとか…。月曜日が減るトレンドとか…。 で、理由（因果）を考える感じになりますよね。

prophet_plot_components(p_model, forecast)

11.10 prophet分析:パラメータの調整

もう少し、データにフィットさせるには、パラメータを調整します。 changepoint.prior.scaleを0.01に変更してみます。

p_model = read_df %>% prophet(
  growth = "linear",
  changepoint.prior.scale = 0.02,

  seasonality.mode = "multiplicative",
  seasonality.prior.scale = 10,
  
  yearly.seasonality = TRUE,
  weekly.seasonality = TRUE,
  daily.seasonality  = FALSE,
  
  holidays = NULL,
  holidays.prior.scale = 10
)

future = make_future_dataframe(p_model, periods = 7)
forecast = predict(p_model, future)
dyplot.prophet(p_model, forecast)
