3.2 Package

触れるグラフを描くなら、「plotly」パッケージです。

library(plotly)

3.3 plotlyのグラフ

plotlyでグラフを描くと下記のようになります。 グラフがグリグリ動かせます♪

plot_ly(airquality,
        x=~Month, 
        y=~Temp, 
        z=~Ozone,
        color=~Ozone, 
        mode= "markers",
        size=8,    
        type = "scatter3d") %>% 
  layout(title = "airquality")
