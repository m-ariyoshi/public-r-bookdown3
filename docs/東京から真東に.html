13 東京から真東に

13.1 はじめに

地球が丸いことを時々忘れる。そんな時は落ち着いて考える。

13.2 Package

今回は、「geosphere」です。

library(leaflet)
library(sf)
library(geosphere)
library(dplyr)

13.3 使用データ

東京駅のポイントを作成します。

tokyo <- c(139.766905, 35.681242)

p_tokyo = st_sfc(st_point(tokyo), crs = 4326)

leaflet() %>%
  addTiles() %>%
  addMarkers(data=p_tokyo) %>% 
  addScaleBar(position = "bottomleft",scaleBarOptions(imperial = FALSE))

13.4 真東に100km進む

なんとなくだけど、北緯が同じところをイメージしますよね。違うんです。 東京駅を基準に、真東（９０°）に、1000km進んでみましょう。

NHKリンク [link] https://www2.nhk.or.jp/school/watch/clip/?das_id=D0005310422_00000

test_dist  = 1000 * 1000
test_angle = 90
ans = c(destPoint(tokyo, test_angle, test_dist))

p_ans   = st_sfc(st_point(ans), crs = 4326)

p_ans_line = st_sfc(st_linestring(rbind(tokyo,ans)), crs = 4326)
p_tokyo_buff = p_tokyo %>% st_transform(crs=32654) %>% 
  st_buffer(dist=test_dist) %>% 
  st_transform(crs=4326)

leaflet() %>%
  addTiles() %>%
  addMarkers(data=p_tokyo) %>% 
  addMarkers(data=p_ans) %>% 
  addPolylines(data=p_ans_line) %>% 
  addPolylines(data=p_tokyo_buff) %>% 
  addScaleBar(position = "bottomleft",scaleBarOptions(imperial = FALSE))
