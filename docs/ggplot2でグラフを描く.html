2 ggplot2でグラフを描く

2.1 使用するデータ

使用するデータは、Rのサンプルデータ「cars」。

summary(cars)
##      speed           dist       
##  Min.   : 4.0   Min.   :  2.00  
##  1st Qu.:12.0   1st Qu.: 26.00  
##  Median :15.0   Median : 36.00  
##  Mean   :15.4   Mean   : 42.98  
##  3rd Qu.:19.0   3rd Qu.: 56.00  
##  Max.   :25.0   Max.   :120.00

2.2 Package

Rでグラフを描くなら、「ggplot2」パッケージです。

library(ggplot2)

2.3 普通のグラフ

RのDefaultでグラフを描くと下記のようになります。

plot(cars)

2.4 ggplot2のグラフ

Rのggplot2でグラフを描くと下記のようになります。

# colnames(cars)
ggplot(cars, aes(x=speed, y=dist)) +
    geom_point(size=3)

2.5 少し装飾すると… 

# Windowsフォントを設定
windowsFonts("MEI"=windowsFont("Meiryo"))
# colnames(cars)
ggplot(cars, aes(x=speed, y=dist)) + 
  geom_point(size=3, color="blue") +
  ggtitle("車のスピードと停止距離") + 
  theme_minimal(base_size = 12, base_family = "MEI")
